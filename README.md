A real-time financial sentiment dashboard analyzing recent news headlines across major sectors
using both FinBERT and VADER NLP models.  
Computes a **Portfolio Sentiment Index (PSI)** and visualizes sector-level sentiment interactively
through Streamlit.

## Features
- Fetches live sector news from NewsAPI
- Runs sentiment analysis with FinBERT (finance-tuned BERT) and VADER
- Computes weighted sentiment & PSI (Portfolio Sentiment Index)
- Includes Z-score normalization and correlation with S&P 500 returns
- Interactive Streamlit dashboard for portfolio weighting

## Tools
Python, Pandas, Streamlit, Transformers, VADER, yFinance, Matplotlib
